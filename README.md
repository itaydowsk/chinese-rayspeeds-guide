# chinese-rayspeeds-guide

本仓库用于归档和发布多个独立的 HTML 页面，内容围绕中国网络环境下与“速度”相关的信息整理与展示。这些页面彼此独立，不针对任何特定域名或具体网站。

## 项目简介

本项目旨在提供一个结构化的静态页面集合，方便用户浏览和查阅与网络速度相关的资料。所有页面均为纯 HTML 格式，无需后端服务即可直接打开使用。

## 目录说明

```
.
├── index.html          # 总索引页，列出所有归档页面
├── pages/              # 存放各个独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（CSS、图片等）
└── README.md           # 本文件
```

- `index.html`：仓库首页，提供页面导航。
- `pages/`：所有独立 HTML 页面存放于此，每个文件代表一个独立主题或数据展示。
- `assets/`：存放页面共用的样式、脚本或图片资源。

## 页面归档说明

- 每个页面均以独立 HTML 文件形式存在，内容完整，可离线打开。
- 页面主题围绕网络速度相关信息的整理与可视化，不包含具体商业推广或使用引导。
- 所有页面均为静态内容，不收集用户信息，不加载外部资源（除非必要）。
- 页面更新时会保留历史版本，便于追溯变化。
